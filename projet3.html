Key Performance

J'ai trouvé cette conférence super intéressante ! Cette conférence démontrais en quoi le temps de chargement d'un site web peut influer sur les utilisateurs et la publicité de celui-ci. Le temps de chargemet peut autant être celui que l'on a à l'arriver sur le site ou encore celui que l'on a lorsque on veut soumettre un formulaire ou changer de page dans le site. Lors de cette conférences plein d'astuce ont été données pour améliorer l'efficacité d'un site web.

Azure by MIC
